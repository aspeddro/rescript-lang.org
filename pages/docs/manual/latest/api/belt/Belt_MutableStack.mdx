### dynamicPopIter
`dynamicPopIter(s, f)` apply `f` to each element of `s`. The item is poped
  before applying `f`, `s` will be empty after this opeartion. This function is
  useful for worklist algorithm.

```res sig
let dynamicPopIter: (t<'a>, 'a => unit) => unit
```
