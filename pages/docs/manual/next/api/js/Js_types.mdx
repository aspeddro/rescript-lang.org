# Js_types

Provide utilities for manipulating JS types.

## symbol

```res sig
type symbol
```
Js symbol type (only available in ES6)

## bigint_val

```res sig
type bigint_val
```

## t

```res sig
type t<_> =
  | Undefined: t<undefined_val>
  | Null: t<null_val>
  | Boolean: t<bool>
  | Number: t<float>
  | String: t<string>
  | Function: t<function_val>
  | Object: t<obj_val>
  | Symbol: t<symbol>
  | BigInt: t<bigint_val>
```


## test

```res sig
let test: ('a, t<'b>) => bool
```
`test(value, t)` returns `true` if `value` is `typeof t`, otherwise `false`.
   This is useful for doing runtime reflection on any given value.

   ```res example
   test("test", String) == true
   test(() => true, Function) == true
   test("test", Boolean) == false
   ```

## tagged_t

```res sig
type tagged_t =

  | JSNull
  | JSUndefined
  | JSNumber(float)
  | JSString(string)
  | JSFunction(function_val)

```
